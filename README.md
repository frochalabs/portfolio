# Portfolio

Portfólio pessoal desenvolvido com Next.js e TypeScript, inspirado em um design moderno e minimalista.

## Tecnologias

- Next.js 14
- TypeScript
- Tailwind CSS
- React 18

## Como executar

1. Instale as dependências:
```bash
npm install
```

2. Execute o servidor de desenvolvimento:
```bash
npm run dev
```

3. Abra [http://localhost:3000](http://localhost:3000) no seu navegador.

## Estrutura do projeto

- `/app` - Páginas e layouts do Next.js
- `/components` - Componentes reutilizáveis
- `/public` - Arquivos estáticos

## Personalização

Edite os componentes em `/components` para personalizar o conteúdo do seu portfólio:
- `Hero.tsx` - Seção inicial
- `About.tsx` - Sobre você
- `Projects.tsx` - Seus projetos
- `Contact.tsx` - Formulário de contato
